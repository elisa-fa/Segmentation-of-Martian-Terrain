# Segmentation-of-Martian-Terrain

## Project Overview
This repository contains the development and results of a Deep Learning project focused on the semantic segmentation of Martian terrain. The goal of the project was to design and implement a robust neural network capable of classifying each pixel of grayscale images into one of five distinct terrain classes: Background, Soil, Bedrock, Sand, and Big Rock.

The project addresses common challenges in remote sensing and autonomous navigation, such as class imbalance and the unstructured nature of planetary landscapes.

## Repository Content
The repository is organized into the following main components:

* MarsTerrainSegmentation_Notebook.ipynb: A comprehensive Jupyter Notebook containing the full end-to-end pipeline. This includes data preprocessing, exploratory data analysis, the implementation of a custom encoder-decoder architecture, model training, and performance evaluation.
* MarsTerrainSegmentation_Report.pdf: A formal technical report detailing the theoretical approach, the experimental setup, architectural choices (such as attention mechanisms and normalization strategies), and a thorough analysis of the achieved results.
